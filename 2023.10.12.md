# chapter8 다중 반복문

## chapter 8(1)

다중 반복문 사용시 유의 사항

각 반복문을 제어하는 제어 변수가 달라야 한다

```
#include <stdio.h>
int main(void)
{
	int i, j;
	for (j = 0; j <= 2; j += 1)
	{
		for (i = 0; i <= 4; i += 1)
			printf("*");
		printf("\n");
	}
	return 0;
}
```

### 실습문제

```
#include <stdio.h>
int main(void)
{
	int i, j;
	for (j = 0; j <= 9; j += 1)
	{
		for (i = 0; i <= 9; i += 1)
			printf("%3d", j*10+i);
		// 3%d 숫자를 보기 좋게 정렬하거나 
		// 고정폭(fixed-width) 형식으로 출력하고자 할 때 유용
		printf("\n");
	}
	return 0;
}
```

### 예제 8-2

```
#include <stdio.h>
int main(void)
{
	int i, j;
	for (j = 2; j <= 9; j += 1)
	{
		printf("=== %d단 ===\n", j);

		for (i = 1; i <= 9; i += 1)
			printf("%2d x %d = %2d", j, i, j*i);
		printf("\n");
	}
	return 0;
}
```

### 의문점

printf("2%d x %d = %2d"); 이 줄에 변수를 입력 안 했는데 왜 실행이 되지...?
```
#include <stdio.h>
int main(void)
{
	int i, j;
	for (j = 2; j <= 9; j += 1)
	{
		printf("=== %d단 ===\n", j);

		for (i = 1; i <= 9; i += 1)
			printf("2%d x %d = %2d");
		printf("\n");
	}
	return 0;
}
```
